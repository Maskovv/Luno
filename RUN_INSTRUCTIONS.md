# Инструкция по запуску приложения

## Шаг 1: Установка зависимостей

Откройте терминал в папке проекта и выполните:

```bash
npm install
```

Это установит все необходимые пакеты (React, Vite, Firebase, и т.д.)

## Шаг 2: Настройка Firebase (обязательно)

Приложение требует настройки Firebase для работы аутентификации.

### Быстрая настройка:

1. Перейдите на https://console.firebase.google.com/
2. Создайте новый проект (или используйте существующий)
3. В настройках проекта найдите "Ваши приложения" → добавьте веб-приложение
4. Скопируйте конфигурацию Firebase
5. Откройте файл `src/shared/config/firebase.js`
6. Замените значения в `firebaseConfig` на ваши данные

### Включение анонимной аутентификации:

1. В Firebase Console перейдите в "Authentication"
2. Нажмите "Начать"
3. Вкладка "Методы входа" → включите "Анонимный вход"

## Шаг 3: Запуск приложения

После установки зависимостей и настройки Firebase:

```bash
npm run dev
```

Приложение запустится и будет доступно по адресу:
- **http://localhost:3000**

## Шаг 4: Проверка работы

1. Откройте браузер и перейдите на **http://localhost:3000/qr**
   - Вы увидите страницу с QR-кодом для подключения учеников

2. Откройте новую вкладку и перейдите на **http://localhost:3000/login**
   - Введите имя и нажмите "Начать игру"
   - Вы попадете на страницу выбора уровня

3. Выберите уровень 1
   - Вы увидите диалоги с персонажем
   - Нажимайте "Далее" для перехода между репликами

4. Проверьте статистику
   - Перейдите на **http://localhost:3000/statistics**
   - Увидите статистику по ученикам

## Возможные проблемы

### Ошибка при запуске `npm install`
- Убедитесь, что у вас установлен Node.js (версия 16 или выше)
- Проверьте: `node --version`

### Ошибка Firebase при входе
- Проверьте, что вы правильно скопировали конфигурацию Firebase
- Убедитесь, что анонимная аутентификация включена в Firebase Console

### Порт 3000 занят
- Измените порт в файле `vite.config.js` (поле `port`)

## Полезные команды

- `npm run dev` - запуск в режиме разработки
- `npm run build` - сборка для продакшена
- `npm run preview` - предпросмотр собранного приложения
