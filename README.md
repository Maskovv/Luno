# Игра по кибербезопасности

Образовательная веб-игра для изучения кибербезопасности.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3000

## Структура проекта

Проект использует feature-first архитектуру:
- `src/features/` - основные фичи приложения
- `src/shared/` - общие компоненты и утилиты
- `src/app/` - конфигурация приложения (роутинг, провайдеры)
