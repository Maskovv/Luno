# Быстрый старт

## Установка зависимостей

```bash
npm install
```

## Настройка Firebase

1. Следуйте инструкциям в файле `FIREBASE_SETUP.md`
2. Обновите конфигурацию в `src/shared/config/firebase.js`

## Запуск приложения

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Использование

1. Откройте `http://localhost:3000/qr` - это страница для учителя с QR-кодом
2. Ученики сканируют QR-код или переходят по ссылке
3. Ученики вводят свое имя и начинают игру
4. Учитель может просматривать статистику на странице `/statistics`

## Структура данных

Прогресс учеников сохраняется в `localStorage`:
- `progress_{userId}` - разблокированные уровни
- `level_{levelId}_{userId}` - прогресс прохождения уровня
- `userName_{userId}` - имя пользователя
